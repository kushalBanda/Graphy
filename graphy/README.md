# Graphy

Vscode Extension: Graphy

Create a markdown file which is a graphical representation of the Codebase with Mermaid Diagrams, UML Diagrams, Architectural Flows (HLD and LLD) and comprehensive documentation that helps AI assistants like Cursor and Claude better understand the codebase.

## Yo Code - Extension and Customization Generator

https://github.com/microsoft/vscode-generator-code

## Features

- Analyzes codebase structure
- Generates Mermaid diagrams
- Creates HLD (High Level Design) and LLD (Low Level Design) documentation
- Produces Graphy.md file for AI assistant integration
